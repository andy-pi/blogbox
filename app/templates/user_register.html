Complete Registration Process

Thankyou for purchasing a subscription to BlogBox. Please enter the following details to complete your registration.

1. If you don't have your own domain name simply enter your domain as mysite.blogbox.com (only numbers 0-9, letters a-z and hyphens can be used, no http:// is needed)

2. If you have your own domain name, enter that instead. You will be able to access your blog at a test URL, to which you will need to point your domain name server to (CNAME record). Developers note this registration process will create an Amazon Web Services S3 bucket named as the domain name you enter here, which must be available for registration to proceed.
