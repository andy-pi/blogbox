Frequently Asked Questions

How do I set up a custom domain name?
If you own a custom domain name, such as example.com, choose example.com as your domain name in BlogBox. On your settings page you will see a longer URL, to which you need to point your DNS (Domain Name Server) CNAME record to.

How do I use a BlogBox subdomain?
Simply change your domain to example.blogbox.com. You don't need to do anything else, we will set up the DNS for you.

What headings should I use in my blog post?
Date: 2013-03-15 20:20
Title: My Blog Post
Slug: my-post
Category: Ramblings
Author: Author Name
Summary: Summary of the blog for the index page

When my blog is published all my text is all one one line? What happened?
To insert a line break / carriage return in Markdown, end each line with TWO spaces and then press enter / carriage return key. If you just end a with a carriage return, when your blog is uploaded it will be ignored and your text will appear on one line. See the Markdown cheatsheet for more information on formatting using Markdown. You'll see the same behaviour in the Dillinger preview window!

How can I add a picture to my blog from Dropbox?
You can insert any image from your BlogBox blog, but it must be available online somewhere. The easiest way to do this is through Dropbox.
1. Copy your image to Dropbox. You can store it in your BlogBox folder if you wish.
2. Next, right click on the image, and in the popup menu, left click "Share Dropbox Link".
3. Paste the link to the image into you blog file, enclosed by parentheses, like this:
(https://www.dropbox.com/s/1wraz68pydu8hdq/andypi-colorfinal.svg?dl=0)
4. At the end of the dropbox link, delete dl=o and replace it with raw=1. Just before the link add an exclamation mark and square brackets (with a name inside if you wish), so it will look something like this:
![image_name](https://www.dropbox.com/s/1wraz68pydu8hdq/andypi-colorfinal.svg?raw=1)

I'm concerned about privacy. Can Blogbox access all of my Dropbox? What about my data?
When you authorize Blogbox on Dropbox, it creates its own folder in Dropbox/Apps/Blogbox. Blogbox only has access to this folder, it cannot modify or even see any other folder. Your payment data is all handled through PayPal, we don't store any of this ourselves. The Blogbox website is encrypted with SSL (see the green lock in the browser bar) so any information, such as you blog settings are passed to us securely. We use an industry standard method to save your password indirectly, so it cannot be recovered by hackers.
